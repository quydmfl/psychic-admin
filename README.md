## Environment local.

Step 1: Install docker on local machine.
```
https://www.docker.com/
```

Step 2: Clone source to local machine.
```
git clone git@github.com:quydmfl/psychic-admin.git
```

Step 3: Go inside source and run command to development.
```
cd psychic-admin
```
And
```
docker-compose up
```
